
物理上的商品和逻辑上的概念要区分开。
往往物理设备可能具有多个职能。
而且可能存在已过时不在生产的设备。

逻辑概念作用于OSI网络模型，

# 交换机  switch

数据链路层设备 只关注MAC地址 不关注IP

有端口 和 MAC地址对应表

# HUB 集线器

# router 路由器


# Gateway 网关
连接不同网段的设备。




商品路由器往往承担了交换机+网关的指责


内网设备

路由器

有WAN口和LAN口，如果取掉WAN口只使用LAN口 路由器is交换机

网关

连接两个网络的设备 可以是路由器
