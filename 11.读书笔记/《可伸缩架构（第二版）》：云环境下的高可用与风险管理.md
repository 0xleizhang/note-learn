书中提的风险模型Excel:

code 编码

system 系统，子系统或模块名称

owner 责任人

description 风险描述

date identified 添加到模型中到日期

likelihood 可能性 高中低

severity 严重性 高中低

mitigation plan 风险缓和计划

state 状态 活跃 已缓和，正在修复，已解决

ETA 风险距离计划解决日期到预估时间

monitoring 是否有监控

triggered 触发计划 ，如果真的发生如何处理它，管理层面的计划

comments 备注

tracking ID 跟踪ID 在缺陷管理中的标识

history 是否发生 时间？频率？
